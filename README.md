## 📄 Overview

**Mathematical.py** is a Python-based command-line calculator that performs basic arithmetic operations on both **integers** and **floating-point numbers**. It also includes additional tools like a personal **profile viewer** and an easy-to-understand **help guide**, making it perfect for beginners learning Python or users who need quick calculations from the terminal.

---

## ✅ Features

- 🎯 Supports both integers and floats (e.g., `5`, `3.14`)
- ➕ Performs addition
- ➖ Performs subtraction
- ✖️ Performs multiplication
- ➗ Performs division (with zero-division check)
- 🧮 Performs modulus (with zero-division check)
- 🧑‍💻 Built-in "View Profile" feature (displays developer info)
- 🆘 Integrated help section (guides user on usage)
- ♻️ Menu-driven interface using modern Python `match-case`
